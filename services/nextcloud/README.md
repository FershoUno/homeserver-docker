# WordPress

Implementación de Nextcloud Local ejecutando:  

~~~
docker-compose up -d
~~~

> La instalación puede demorar dependiendo la velocidad de internet y los recursos de hardware que tenga.

Ingrese esta URL en el navegador:
~~~
http://localhost:8000
~~~
